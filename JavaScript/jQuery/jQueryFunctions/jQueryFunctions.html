.click

Click the button to produce an effect!

.hide

If you click this button, I'll hide!

.show

Click this button and the paragraph above will come back.

.toggle

This button just toggles this paragraph.

.slideDown

Click this button to reveal a hidden, slidey sentence.

Sliding in!

.slideUp

This one makes the last box's hidden sentence slide away.

.slideToggle

I toggle up and down when you click this button.

.fadeIn

Click this button to reveal a hidden sentence.

Fading in!

.fadeOut

This button banishes the fading sentence in the last div.

.addClass

This button will add a class to this paragraph, which class will turn it blue!

.before

This button will add stuff above the heading of this div.

.after

This button will add stuff right after the heading.

.append

This button will append an element after this paragraph.

.html

This button will change the html of this whole div. You'll have to refresh the page to get it back.

.attr

This will change the style attribute of this paragraph.

.val

This one is a getter method, so I'm not sure what effect to give the button.

.text

This button will change just the text of this sentence!

.data

This is just going to store arbitrary data in association with this sentence's tag.
